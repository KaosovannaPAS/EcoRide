# Charte Graphique Officielle - EcoRide 🍃

Cette charte est extraite strictement du site en production (https://eco-ride-nu.vercel.app/).

## 🎨 Palette de Couleurs Réelle

| Variable CSS | Couleur | Code Hex | Usage |
| :--- | :--- | :--- | :--- |
| `--eco-primary` | Deep Modern Teal | `#054752` | Header, Footer, Titres, Boutons principaux |
| `--eco-accent` | Vibrant Green | `#00cc66` | Actions Éco, Badges, Hover accent |
| `--eco-bg` | Airy Gray | `#f9f9f9` | Arrière-plan général |
| `--eco-surface`| Pure White | `#ffffff` | Cartes, Sections de contenu |
| `Sticky Bar` | Or/Vert | Gradient | Barre d'offre spéciale (Header) |

## 📐 Typographie & Structure
- **Police** : `Inter` (Sans-serif moderne).
- **Arrondis** : `16px` (lg) pour les cartes et `24px` (xl) pour les boutons et barres de recherche.
- **Ombres** : `shadow-soft` (0 2px 12px rgba(0,0,0,0.08)).

## 🚀 Composants de Référence
- **Header** : Fond blanc avec logo Deep Teal. Menu centré ou à droite.
- **Cartes Trajet** : Titres Teal, prix Vert, badges Électriques (🍃 Écologique).
- **Sticky Bar** : Gradient `#ffcc00` vers `#00cc66` pour les promotions.
